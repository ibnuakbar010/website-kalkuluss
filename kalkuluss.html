Konsep Limit

Limit adalah nilai yang didekati oleh suatu fungsi ketika variabel inputnya mendekati suatu titik tertentu. Konsep ini fundamental dalam kalkulus dan digunakan untuk mendefinisikan turunan dan integral.

Contoh Soal:

Hitunglah: limx→2 (x2 - 4)/(x - 2)

Penyelesaian:

limx→2 (x2 - 4)/(x - 2) = limx→2 (x + 2)(x - 2)/(x - 2)

= limx→2 (x + 2) = 4

Teorema Limit Dasar:

  • limx→a k = k (k konstan)
  • limx→a [f(x) ± g(x)] = lim f(x) ± lim g(x)
  • limx→a [f(x) × g(x)] = lim f(x) × lim g(x)
  • limx→a [f(x)/g(x)] = lim f(x)/lim g(x), jika lim g(x) ≠ 0

Konsep Turunan

Turunan fungsi adalah ukuran seberapa cepat nilai fungsi berubah terhadap perubahan variabel inputnya. Turunan didefinisikan sebagai limit dari rasio perubahan.

Contoh Soal:

Tentukan turunan dari f(x) = 4x3 - 2x2 + 5x - 1

Penyelesaian:

f'(x) = d/dx (4x3) - d/dx (2x2) + d/dx (5x) - d/dx (1)

= 12x2 - 4x + 5

Aturan Dasar Turunan:

  • Aturan Pangkat: d/dx (xn) = nxn-1
  • Aturan Konstan: d/dx (c) = 0
  • Aturan Penjumlahan: d/dx [f(x) + g(x)] = f'(x) + g'(x)
  • Aturan Perkalian: d/dx [f(x)g(x)] = f'(x)g(x) + f(x)g'(x)

Konsep Integral

Integral adalah operasi kebalikan dari turunan. Integral digunakan untuk menghitung luas di bawah kurva, volume benda putar, dan berbagai aplikasi lainnya.

Contoh Soal:

Hitunglah ∫(3x2 + 2x - 5) dx

Penyelesaian:

∫(3x2 + 2x - 5) dx = 3∫x2 dx + 2∫x dx - 5∫1 dx

= x3 + x2 - 5x + C

Teknik Integral Dasar:

  • Integral Tak Tentu: ∫f(x) dx = F(x) + C
  • Integral Tentu: ∫ab f(x) dx = F(b) - F(a)
  • Integral Substitusi: ∫f(g(x))g'(x) dx = ∫f(u) du
  • Integral Parsial: ∫u dv = uv - ∫v du

Latihan Soal

Uji pemahaman Anda dengan berbagai soal kalkulus dari berbagai tingkat kesulitan.

Limit Level: Mudah

Soal 1

Hitunglah: limx→3 (x2 - 9)/(x - 3)

Pembahasan:

Faktorkan pembilang:

(x2 - 9) = (x + 3)(x - 3)

limx→3 (x + 3)(x - 3)/(x - 3) = limx→3 (x + 3)

= 6

Turunan Level: Sedang

Soal 2

Tentukan turunan pertama dari f(x) = (2x + 1)(x2 - 3)

Pembahasan:

Gunakan aturan perkalian:

f'(x) = (2)(x2 - 3) + (2x + 1)(2x)

= 2x2 - 6 + 4x2 + 2x

= 6x2 + 2x - 6

Integral Level: Sedang

Soal 3

Hitunglah ∫(3x2 + 4x - 2) dx

Pembahasan:

Integralkan setiap suku:

∫3x2 dx = x3

∫4x dx = 2x2

∫-2 dx = -2x

= x3 + 2x2 - 2x + C

Aplikasi Level: Sulit

Soal 4

Sebuah bola dilempar ke atas dengan kecepatan awal 20 m/s. Tinggi bola setelah t detik dinyatakan dengan h(t) = 20t - 5t2. Kapan bola mencapai tinggi maksimum?

Pembahasan:

Tinggi maksimum terjadi ketika turunan h(t) = 0:

h'(t) = 20 - 10t

20 - 10t = 0 → t = 2 detik

Jadi bola mencapai tinggi maksimum setelah 2 detik.
